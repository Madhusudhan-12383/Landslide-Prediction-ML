# Landslide Prediction Using Machine Learning

## 📌 Project Overview
This project predicts landslide occurrence using machine learning techniques based on environmental factors like rainfall, slope, soil type, and vegetation.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 🤖 Model Used
- Random Forest Classifier

## 📊 Features
- Data preprocessing
- Model training & evaluation
- Landslide prediction system

## ▶️ How to Run
1. Open Google Colab
2. Upload the notebook
3. Run all cells

## 📈 Output
Predicts whether a landslide will occur (1 = Yes, 0 = No)
